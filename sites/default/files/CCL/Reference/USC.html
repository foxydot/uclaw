The United States Code is available in a number of different printed and online versions. Printed versions are available at all U.S. despositaries (make this a link). There is an ongoing effort to revise and codify all Titles of the code into positive law. As of June 25, 2003 , Title 15 is still primia facia law. It should be noted that slight and infrequent wording differnces exist among the different versions from their revision and codification into that verision of the code.

 

Printed versions include:

 

The United States Code (U.S.C.)

This has been published as a set every 6 years by the Government Printing Office with the assistance of West Publishing Company. Last published in 2000, it was first published in 1926. While not having all the information found in the following two versions, it has the advantage of having all the relevant securities laws in one volume (Title 15).

 

The United States Code Annoted (U.S.C.A.)

This is published on an ongoing basis by West Publishing Company. The back of each volume contains a "Cumulative Annual Pocket Part" which contains all changes to the material in that volume. Generally when this insert becomes to bulky to be contained in the volume a new volume is published. This version of the code contains annotations from State and Federal Courts.

 

The United States Code Service (U.S.C.S.)

This is a set published by Lexis Law Publishing with an annual cumulative supplement in the back.
